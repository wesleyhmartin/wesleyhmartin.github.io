2.0.0

New keyboard shortcuts:

Press + for maximum speed, - for minimum speed.

New Features:

Change default volume and speed in the Options page.

Now a time display instead of a progress bar. Press to go to a specific duration.

Requires fewer permissions.

Bug Fixes:

Don't announce value twice when changing sliders.

[, ], +, -, =, and Space now work when focus is on a slider.
